Colour Encoding for IoT Device Configuration

Click the "Writer" button on one device, then load this page on another device which has a webcam and point the first device at the camera of the second such that the 9 colour squares completely fill the camera view. The message entered on the writer device will be transmitted to the reader device.

N.B. In reader mode you should see raw bits straight away, but you might have to wait 10s or so before raw bytes start appearing. This is because the code is waiting for the next start-of-message marker to come round.
